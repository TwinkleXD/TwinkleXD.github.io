跨域问题总结

出现原因：

浏览器的同源策略

  • 同源策略
    事实上，同源策略是浏览器最基本的一种安全机制，能规避一定程度的风险，主要是来自接口的请求和Dom的查询这两个方面的。

同源策略用于限制同源文档或脚本与另一个源的资源进行交互。它能帮助阻隔恶意文档，减少可能被攻击的媒介。
by MDN Web Docs 浏览器的同源策略

同源指两个URL的 协议/主机(/端口)都相同，这两个URL才是同源的，路径可以不同。
