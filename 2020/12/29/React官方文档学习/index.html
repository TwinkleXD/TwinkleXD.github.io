React官方文档学习

环境

前提准备

  • Node.js (>=8.10)
  • npm (>=5.6)

创建项目

核心概念

JSX

JavaScript 的语法扩展

  1. 可以直接在js里写标签
  2. 标签内可以在大括号中放任何有效的JS表达式
  3. 标签属性可以通过引号或者大括号指定
  4. JSX直接使用是安全的（React DOM在渲染输入内容之前会转义成字符串，有效地防止XSS（cross-site-scripting, 跨站脚本）攻击）
  5. React元素：描述显示内容，是普通对象，不可变
    1. Babel把JSX转译成React.createElement()函数调用
    2. React.createElement()预先执行检查，创建对象
  • JSX –> Babel转译 –> 实际创建元素

    《React 官方文档》

元素渲染

  1. React DOM负责更新浏览器DOM与React保持一致
  2. ReactDOM.render()将React元素渲染到根DOM节点中
  3. 当前更新UI的方式：创建全新元素传入ReactDOM.render()
  4. React DOM会将元素和它的子元素与它们之前的状态进行比较，只更新需要更新的部分
